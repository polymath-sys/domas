Domas — One-Page Summary

What it is: React SPA using Firebase Auth and Firestore (realtime) to manage user profiles and announcements.

Key flows

  • Registration: createUserWithEmailAndPassword -> setDoc(user/{uid}).
  • Login: signInWithEmailAndPassword, tracked via useAuthState(auth).
  • Profile: getDoc / updateDoc on user/{uid}.
  • Announcements: onSnapshot listener on announcements for realtime updates.

Files to show

  • src/firebase.jsx
  • src/Register.jsx
  • src/components/Profile/Profilepage.jsx
  • src/components/Rightsidebar.jsx

Quick debug checklist

  • Check Firestore rules for `permission-denied`.
  • Disable ad-blockers for `localhost` for `ERR_BLOCKED_BY_CLIENT` errors.
  • Verify `firebaseConfig` matches Project Settings.

Demo steps

  1. Register a new user and show Firestore `user/{uid}` document.
  2. Create an announcement from another browser or console.
  3. Show right sidebar updating in real time.
